# Rust Exercise

This project contains small Rust exercises

## Doing exercises

Running:

```bash
cargo run -p <EXERCISE_NAME>
```

Tests:

```bash
cargo test -p <EXERCISE_NAME>
```

Lints:

```bash
cargo clippy -p <EXERCISE_NAME>
```

Doing a single exercise:

```bash
cargo run
```

```bash
cargo test
```

```bash
cargo clippy
```
